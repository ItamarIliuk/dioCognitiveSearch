# Azure Cognitive Search com Azure AI

Este repositório apresenta os passos principais para a utilização do serviço de **Cognitive Search** do Azure AI, com base no tutorial disponível [aqui](https://microsoftlearning.github.io/mslearn-ai-fundamentals/Instructions/Labs/11-ai-search.html). Seja você uma empresa, um escritório de advocacia ou qualquer outra organização que necessite de uma solução de busca inteligente, este guia ajudará a criar aplicações que melhoram a experiência de pesquisa e o acesso a informações críticas.

## Descrição

Com o Azure Cognitive Search, é possível:

- **Criar índices inteligentes:** Organize e armazene seus dados de forma estruturada para permitir buscas eficientes.
- **Enriquecer dados automaticamente:** Utilize skillsets para transformar e extrair insights dos seus dados (como análise de linguagem, detecção de entidades, tradução e muito mais).
- **Realizar consultas avançadas:** Implemente filtros, facetas e classificações personalizadas para encontrar rapidamente os dados desejados.
- **Integrar com outras ferramentas e aplicações:** Conecte sua aplicação web, mobile ou sistemas corporativos a um serviço robusto e escalável de busca.

## Possibilidades de Uso

- **Empresas:** Melhore a experiência do usuário em portais e intranets, facilitando a busca por documentos, produtos e informações corporativas.
- **Escritórios de Advocacia:** Organize e agilize a localização de contratos, jurisprudências e documentos legais, acelerando a tomada de decisões e a pesquisa jurídica.
- **Setores Diversos:** Ideal para editoras, instituições de ensino, agências de marketing, entre outros, onde a busca por informações precisas é essencial para a produtividade e a eficiência.

## Conteúdo do Tutorial

O tutorial disponibilizado no link acima cobre os seguintes tópicos:

1. **Criação do Serviço e Configuração Inicial:**  
   - Provisionamento do serviço de Cognitive Search no Azure.
   - Configuração inicial pelo portal do Azure.

2. **Criação e Gerenciamento de Índices:**  
   - Definição da estrutura dos índices.
   - Estratégias para armazenar e organizar os dados de maneira eficiente.

3. **Processamento de Dados com Skillsets:**  
   - Aplicação de transformações aos dados (extração de metadados, análise de texto, detecção de idioma, entre outros).
   - Enriquecimento dos dados para melhorar a relevância dos resultados de busca.

4. **Consultas Personalizadas e Uso da REST API:**  
   - Exemplos práticos de como realizar pesquisas avançadas.
   - Integração via REST API para extrair dados relevantes conforme os critérios definidos.

5. **Integração com Aplicações:**  
   - Orientações de como incorporar a busca inteligente em aplicações web e mobile.
   - Melhoria na experiência do usuário ao acessar informações de forma rápida e intuitiva.

## Requisitos

- Conta no Azure com acesso ao serviço de Cognitive Search.
- Familiaridade com o portal do Azure e conceitos básicos de serviços cognitivos.
- Conhecimentos básicos de REST API e manipulação de dados contribuirão para uma melhor compreensão do tutorial.

## Como Utilizar Este Repositório

   - Explore diferentes cenários de pesquisa.
   - Adapte o código conforme as necessidades da sua organização.
